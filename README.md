# Vasya Uppercase / Lowercase Problem

A simple C++ solution to decide whether a word should be converted to **uppercase** or **lowercase** based on letter count.

## Problem Rule

* If uppercase letters are more than lowercase → convert to **UPPERCASE**
* Otherwise (lowercase বেশি বা সমান হলে) → convert to **lowercase**

## Examples

* `HoUse` → `house`
* `ViP` → `VIP`
* `maTRIx` → `matrix`

## Approach

* Count uppercase and lowercase letters using a loop
* Convert the entire string based on the comparison
